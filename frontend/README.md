# Медичний асистент — фронтенд

React-додаток для чату з медичним асистентом: пошук ліків за фото (Find) та питання текстом (Ask). Підтримує логін через Google та список розмов у сайдбарі.

---

## Опис

- **Стек:** React 18, TypeScript, Vite, Tailwind CSS.
- **Функції:** логін через Google (редирект на бекенд), сайдбар з розмовами (нова / вибір / видалення), перемикач режимів Find / Ask (dropdown), відправка повідомлень та фото через API бекенду.

---

## Вимоги

- **Node.js** 18 або новіший
- **npm** (або yarn / pnpm)
- Працюючий **бекенд** на `http://127.0.0.1:8000` (див. кореневий README проєкту)

---

## Встановлення

```bash
cd frontend
npm install
```

---

## Запуск

### Режим розробки

```bash
npm run dev
```

- Додаток: **http://localhost:5173**
- Запити до API проксуються з `/api` на `http://127.0.0.1:8000` (налаштовано в `vite.config.ts`).

Перед відкриттям сторінки переконайтесь, що бекенд запущений.

### Збірка для продакшену

```bash
npm run build
```

Результат у папці `dist/`. Для перегляду збірки локально:

```bash
npm run preview
```

---

## Змінні середовища

У режимі розробки проксі до бекенду вже налаштований. Якщо фронтенд запускати на іншому порту або з іншої домени, при потребі можна додати у проєкт змінні Vite (наприклад `VITE_API_BASE`), але за замовчуванням достатньо налаштувань у `vite.config.ts` та на бекенді (`FRONTEND_URL`, `API_BASE_URL`).
